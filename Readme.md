# Brand Profit Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes brand performance, revenue, profit, units sold, and product insights across different paint brands.

## Objectives
- Analyze revenue and profit trends
- Compare brand performance
- Identify top-performing products
- Track profit margins
- Generate business insights

## Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## Dashboard Features
- KPI Cards
- Brand-wise Profit Analysis
- Revenue vs Profit Comparison
- Interactive Filters
- Product Analysis

## Key Insights
- Own brand generated highest total profit
- JSW Paints achieved highest profit margin
- JSW Shield is the top-performing product
